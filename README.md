
# Just Do It

A simple web application that would help you organize your day based on the schedule and context you give. This application utilizes Cloudflare's Workers AI to help in creating the schedule that you would need.

---

## Tech Used

Typescript JS alongside Tailwind CSS was used to create the front-end while Node JS was used for sending and receiving the request from the AI API. Cloudflare's Workers AI is used for the AI output and input.
